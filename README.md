# 🎬 나만의 영화 소개 웹 사이트

우아한 테크코스 8기 오픈미션 과제

## 📌 프로젝트 소개

TMDB API를 활용한 영화 정보 제공 웹 사이트입니다.

## 🛠️ 기술 스택 (익숙하지 않는 기술들을 활용)

- **React** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 스타일링
- **Github Pages** - 배포 플랫폼
- **GitHub Actions** - CI/CD

## ✨ 주요 기능

### 영화 목록

- [ ] 현재 상영중인 영화 목록
- [ ] 인기 영화 목록
- [ ] 개봉 예정 영화 목록
- [ ] 최고 평점 영화 목록

### 검색 & 상세

- [ ] 영화 제목 검색
- [ ] 영화 상세 정보 조회

### 배포 & 자동화

- [x] Github Pages 배포
- [x] GitHub Actions CI/CD 파이프라인
